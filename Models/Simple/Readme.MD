# **UwU30 Simple Case**

### I have not printed this version so I cannot guarentee it will work, though everything points to it working

### Notes
- This version requires minimal support material due to the removal of the lip on the bottom of the top case
- Supports are still needed for the top case
- No supports are needed for the badge or the case bottom
- The front lip will still need some supports under it, but as it's on the underside, it doesn't need to be perfect
- Any AMS/multicolor options have been removed and are now extruded into the case/badge


### Differences from complex version
- Changed multi body badge into an engraved version so it can be printed in 1 color
