# **UwU30 Complex Case**

### Notes
- This version requires *a lot* of support material due to a lip that I wanted on the base of the case
- ![Base support example](https://github.com/DashDashUnderscoreDash/Thick-Plate-Guide/blob/main/Images/Minisub%20KLE.png)
- The top of the case was printed in this orientation because if you were to print it top down you'd be dealing with a nightmare of supports for the long front lip
- It is HIGHLY encouraged to use [Bambu's dedicated support material](https://us.store.bambulab.com/products/support-for-pla-petg) for the interface layer. This is the only way I've printed this case and I imagine it's a nightmare to use normal supports
- The example images were all printed in PLA at 0.2mm layour height on a Bambu P1S with a 0.4mm nozzle
