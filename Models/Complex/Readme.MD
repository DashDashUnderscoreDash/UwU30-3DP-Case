# **UwU30 Complex Case**

### Notes
- This version requires *a lot* of support material due to a lip that I wanted on the base of the case
- ![Bottom lip example](https://github.com/DashDashUnderscoreDash/UwU30-3DP-Case/blob/main/Models/Complex/Images/Support%20Example%202.jpg?raw=true)
- ![Base support example](https://github.com/DashDashUnderscoreDash/UwU30-3DP-Case/blob/main/Models/Complex/Images/Support%20Example%201.jpg?raw=true)
- The top of the case was printed in this orientation because if you were to print it top down you'd be dealing with a nightmare of supports for the long front lip
- It is HIGHLY encouraged to use [Bambu's dedicated support material](https://us.store.bambulab.com/products/support-for-pla-petg) for the interface layer. This is the only way I've printed this case and I imagine it's a nightmare to use normal supports
- The example images were all printed in PLA at 0.2mm layour height on a Bambu P1S with a 0.4mm nozzle
- Using supports on the SKUF cutouts is not needed, but it gives you a higher quality print when done
- ![Bottom example](https://github.com/DashDashUnderscoreDash/UwU30-3DP-Case/blob/main/Models/Complex/Images/Support%20Example%203.jpg?raw=true)
- The badge is printed face down so the multicolor looks the best. No supports are needed
- ![Badge example 1](https://github.com/DashDashUnderscoreDash/UwU30-3DP-Case/blob/main/Models/Complex/Images/Support%20Example%204.jpg?raw=true)
- ![Badge example 2](https://github.com/DashDashUnderscoreDash/UwU30-3DP-Case/blob/main/Models/Complex/Images/Support%20Example%205.jpg?raw=true)
